# CTF Simulation
This Web app provides interactive online simulation of the [contrast transfer function (CTF)](https://en.wikipedia.org/wiki/Contrast_transfer_function) of transmission electron microscopes.

Click one of these hosting links (<a href="https://ctf-simulation.herokuapp.com">Host 1</a> | <a href="https://share.streamlit.io/wjiang/ctf-simulation/main/ctf_simulation.py">Host 2</a>) to explore CTF function!</a>
